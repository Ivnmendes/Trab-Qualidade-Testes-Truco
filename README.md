## Descrição

Este trabalho tem como objetivo desenvolver testes para o jogo de truco fornecido pelo professor, utilizando as técnicas de teste abordadas na disciplina de qualidade de software.

## Tecnologias

- Linguagem de programação: Python
- Framework de testes: pytest

## Como executar os testes

1. Clone o repositório
2. Crie um ambiente virtual
3. Ative o ambiente virtual
4. Instale as dependências
5. Execute os testes

```bash
git clone https://github.com/Ivnmendes/Trab-Qualidade-Testes-Truco.git
cd Trab-Qualidade-Testes-Truco
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pytest
```

Para rodar o jogo execute `python updated_main.py` dentro do ambiente virtual.
